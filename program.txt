При разработке проекта использовались следующие библиотеки
tkinter
sqlite3
xlsxwriter

Программа, манипулирующая данными из бд реализована в виде основного окна, где 
можно выбрать вкладки для дальнейшей работы с данными и дочерних, где содератся способы реализации процедур взаимодействия с данными.

Программа имеет следующие вкладки:
1) Добавить – производится добавление данных в БД*.
2)Редактировать – производится редактирование данных по выбранному id.
3)Удалить – обновление флага Удален на “Да” по выбранному id.
4) Просмотреть действующих клиентов – возможность просмотра пользователей, чье значение в столбце Удале равно значению “Нет”
5)Обновить – показ всех джанных из БД
6)Экспорт в файл формата .xlsx


Данная программа требует доработки, реализовав поиск данных, возможность иморта данных в БД из cvs, xlsx.



* - если БД нет, она создается автоматически с названием task.db
